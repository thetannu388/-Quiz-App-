Quiz App

A simple and interactive Quiz Application built using HTML, CSS, and JavaScript. Users can answer multiple-choice questions and view their final score at the end of the quiz.

Features

- Multiple-choice questions
- Instant answer selection
- Score calculation
- User-friendly interface
- Responsive design

Technologies Used

- HTML
- CSS
- JavaScript

How to Use

1. Open the quiz application.
2. Read each question carefully.
3. Select the correct answer.
4. Complete all questions.
5. View your final score.

Learning Outcome

This project helped me practice:

- DOM Manipulation
- Event Handling
- Arrays and Objects
- Conditional Statements
- JavaScript Functions
